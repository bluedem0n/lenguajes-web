Algunos lenguajes de la web

HTML + CSS + Javascript

  • HTML 5

    • HTML5 (Hyper Text Markup Language, versión 5) es la quinta revision importante del lenguaje basico de la World Wide Web. HTML.

    Usa la extensión.html

  • CSS 3

    • Hoja de estilo en cascada o CSS (siglas en inglés de casacading style sheets) es un lenguaje usado para definir y crear la presentacion de un documento estructurado escrito en HTML.

    Usa la extensión css

  • Javascript

    • JavaScript (abreviado comunmente) "JS" es un lenguaje de programacion interpretado utilizado principalmente en el lado del cliente (client-side) ,implementando como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas.

    Usa la extensión.js
